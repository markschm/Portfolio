Projects

Stonk Bot | Hackathon | 2022

A stock trading simulator in Discord using Python, Discord API, and Financial Modeling Prep API. Users could manage their virtual portfolios, buy and sell shares using real-time stock prices, and view their ranking on a server-wide leaderboard.

Personal Portfolio Builder | Hackathon | 2022

Personal portfolio builder, to help computer science students without web design skills create professional-looking personal websites to showcase their skills and experience to potential employers. The program used HTML, CSS and PHP and included a range of prebuilt templates that users could choose from and customize with their own information. Upon completion, users were provided with the HTML and CSS files for their chosen template, with their personal data already inserted.

MLB Data Explorer | 2023

For this project, I plan to create a program that allows users to explore and analyze various baseball statistics. The program will include a database of baseball statistics that is stored using SQLite, as well as a user interface that will allow users to query the database to retrieve statistics on specific teams and players. Additionally, I will add a way for users to plot and visualize particular statistics from the data set.
